## ✅ commit Convention
- commit message 에는 문제 식별자와 문제 난이도를 **필수** 로 기입해주세요.!
- 문제 해결을 위해 접근했던 방법들(Apporaches) 과 문제를 해결할 수 있었던 결정적인 생각(Main Idea), 비슷한 유형의 문제를 만났을 때 상기해야할 점(Impressions)에 대해 적어보세요.!
  
#### _commit 메세지 템플릿_
```
{플랫폼 명}_{문제번호}_{문제이름}/{문제 난이도}

# Apporaches (Optional)

# Main Idea (Optional)

# Impressions (Optional)

```
